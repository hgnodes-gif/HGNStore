# MCFleet-जैसी Store — GitHub Starter Template यह दस्तावेज़ एक पूर्ण *single-repo* starter template देता है जिससे आप **MCFleet जैसी Minecraft store / hosting storefront** GitHub पर बना कर GitHub Pages से होस्ट कर सकते हैं। नीचे दिए गए फाइल्स और दिशा-निर्देश सीधे कॉपी-पेस्ट करने के लिए तैयार हैं। --- ## फ़ाइलें (क्या क्या मिलेगा) 1. `index.html` — मुख्य पेज (लैंडिंग + محصولات/प्लान्स कार्ड, navbar, footer) 2. `styles.css` — सरल, responsive CSS (Tailwind न इस्तेमाल किया गया ताकि GitHub Pages पर सीधा चले) 3. `script.js` — UI interaction: mobile menu, cart mockup, form handling 4. `README.md` — Repo setup और deploy करने के स्टेप्स --- ## Quick usage (तेजी से शुरू करने के लिए) 1. नया GitHub repo बनाएं (नाम: `mcfleet-store` या जो चाहें)। 2. नीचे दी गई फाइलें अपने लोकल मशीन पर एक फोल्डर में बना लें और `git init` करके commit कर दें। 3. GitHub पर remote add/push करें। 4. Repo → Settings → Pages → `gh-pages` या `main` branch (root) select करके GitHub Pages enable करें। > पेज live होने में कुछ मिनट लग सकते हैं — GitHub Pages URL: `https://.github.io//` --- ## कस्टमाइज़ेशन सुझाव * Payment: Tebex/Stripe/Bambora इंटीग्रेशन के लिए buy buttons embed करें। * Backend: अगर order management चाहिए तो GitHub repo से अलग small server (Vercel/Netlify/AWS) जोड़ें या Google Forms + Zapier का उपयोग कर रखें। * Product catalogue: JSON फ़ाइल में रखकर JS से लोड कर सकते हैं ताकि product जोड़ना आसान हो। * Domain: GitHub Pages के साथ custom domain जोड़ें (DNS में CNAME जोड़ना होगा) --- ## Files: (copy these into files in your repo) ### index.html ```html HG Network Store — MCFleet Style
